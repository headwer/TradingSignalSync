TradingView Webhook JSON Format

Use this JSON format in your TradingView alerts:

{
    "action": "BUY",
    "symbol": "ETHUSDC",
    "quantity": 0.01,
    "price": "2500.00",
    "message": "TradingView Alert"
}
Parameters:
  • action: "BUY" or "SELL"
  • symbol: Trading pair (default: ETHUSDC)
  • quantity: Trade size (optional, uses default from settings)
  • price: Current price (for reference)
  • message: Optional message
